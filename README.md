# Python Workout: The Path to Mastery

This repository is dedicated to a rigorous practice of Python 3.12+ fundamentals. The goal is to build muscle memory for syntax, type hinting, and clean code principles before advancing to complex architectures.

## 🏗️ Repository Structure
- `/easy`: 50 fundamental exercises focused on core syntax and logic.
- `/tests`: Pytest suites for every exercise.

## 🛠️ Tech Stack
- **Language:** Python 3.12+
- **Linter/Formatting:** PEP 8 compliance.
- **Testing:** Pytest.
- **Typing:** Mandatory `typing` module usage.
